# 📅 Day 14 — Custom Hook

## 📌 What I Learned Today

Today I learned **Custom Hooks** in React.
Custom Hooks are used to create **reusable logic** that can be shared across multiple components.

---

## 🎯 Concepts Covered

* What is a Custom Hook
* Why Custom Hooks are used
* Creating a custom hook
* Naming convention (`use` prefix)
* Reusing logic across components
* Using built-in hooks inside custom hooks (`useState`, `useEffect`)

---

## 🧠 Key Points

* Custom Hook is just a JavaScript function
* Must start with `use` (e.g., `useFetch`, `useToggle`)
* Helps to reuse logic instead of repeating code
* Makes code clean and maintainable
